# ESP32-Juego-Reflejos-Simon
Juego de Reflejos y Simon Dice para ESP32 desarrollado en MicroPython. Incluye modo de 1 y 2 jugadores con medición de tiempo de reacción y sistema de puntuación, además de un modo Simon Dice que evalúa memoria mediante secuencias aleatorias. Integra LEDs, buzzer y botones, aplicando conceptos de sistemas embebidos y programación estructurada.
